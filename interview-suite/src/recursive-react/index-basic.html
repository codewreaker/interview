📁 Recursive File Explorer BASIC

⏱️ Expected time: 15-20 minutes

📋 Your Task

Build a simple file explorer that shows folders and files in a tree structure.

Requirements (just 3!)

  • Show files with 📄 icon, folders with 📁 icon
  • Click a folder to show/hide its contents
  • Indent children to show the hierarchy

Data Structure

interface FileNode {
  name: string;
  type: 'file' | 'folder';
  children?: FileNode[];  // Only folders have children
}
